Skill

Projek

Project 1 - Website Artikel

Website ini saya buat untuk belajar cara menyusun konten tulisan agar terlihat rapi dan terstruktur. Di sini saya fokus menggunakan HTML untuk bagian kerangka dan CSS untuk mengatur tampilan teks serta jarak antar elemen, supaya artikelnya enak dibaca dan tidak membingungkan pengunjung.

Project 2 - Website Portfolio

Proyek ini adalah sebuah website perkenalan diri yang saya bangun dengan bantuan framework Bootstrap. Tujuannya supaya tampilan website bisa otomatis menyesuaikan diri saat dibuka di HP maupun laptop, sehingga proses pengerjaannya jadi lebih cepat namun hasilnya tetap terlihat modern.

Project 3 - Website Galeri

Di proyek ini, saya membuat sebuah halaman untuk menampilkan koleksi buku dalam bentuk galeri foto. Saya belajar cara mengatur posisi gambar agar tersusun sejajar dan rapi menggunakan HTML dan CSS, sehingga siapa pun yang melihatnya bisa dengan mudah mencari judul buku yang diinginkan.
